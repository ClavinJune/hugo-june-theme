Hugo June Theme

Hugo June is a theme that created to serve multilingual personal site and blog. The design itself is created by Jessi and implemented by June. By default, Hugo June has supported English, Indonesian, and Japanese. All the configuration could be modified by adjusting the built-in config file, i18n directory, and data directory.

Features

  • Skip to Content Button
  • Multilingual
  • Google Site Verification Meta Tag
  • Web Monetization Meta Tag
  • Brave Rewards Verification File
  • Deferred CSS
  • Deferred Google Analytics
